1. **Push to GitHub:**
   ```bash
   cd /Users/sunilkpathak/personal/startup/bulkmates/BulkShareApp
   git add docs/
   git commit -m "Add legal pages for App Store submission"
   git push origin main
   ```

2. **Enable GitHub Pages:**
   - Go to: https://github.com/sunilkishorpathak/BulkShareApp/settings/pages
   - Under "Source", select: **Deploy from a branch**
   - Under "Branch", select: **main** and **/docs** folder
   - Click **Save**

3. **Wait 1-2 minutes** for GitHub to build and deploy

4. **Your URLs will be:**
   - Landing Page: `https://sunilkishorpathak.github.io/BulkShareApp/`
   - Privacy Policy: `https://sunilkishorpathak.github.io/BulkShareApp/privacy-policy.html`
   - Terms of Service: `https://sunilkishorpathak.github.io/BulkShareApp/terms-of-service.html`
