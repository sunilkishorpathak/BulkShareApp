## üéØ TEXT OVERLAY DESIGN TIPS

### Font Choices
- **Headline:** SF Pro Display Bold (60-80pt)
- **Subheadline:** SF Pro Text Regular (40-50pt)
- **Color:** White text with dark semi-transparent background
- **Alignment:** Center or left-aligned

### Placement
- **Top third** of screenshot (avoid covering key UI)
- Or **bottom third** if UI is in top half
- Leave 10% margin on all sides

### Background Overlay
```
Semi-transparent gradient:
- Start: rgba(0, 0, 0, 0.7)
- End: rgba(0, 0, 0, 0.3)
```

## üõ†Ô∏è HOW TO CREATE SCREENSHOTS

### Method 1: Using iPhone Simulator (Easiest)

**Steps:**
1. Open Xcode
2. Select "iPhone 15 Pro Max" simulator
3. Run BulkMates app (Cmd + R)
4. Navigate to desired screen
5. Take screenshot: Cmd + S
6. Screenshots saved to Desktop

**Pros:** Free, exact device size
**Cons:** Need to add sample data first

---

### Method 2: Using Real Device

**Steps:**
1. Run app on iPhone 15 Pro Max (if available)
2. Navigate to desired screen
3. Take screenshot: Power + Volume Up
4. AirDrop to Mac

**Pros:** Shows real performance
**Cons:** Need specific device, manual cropping

---

### Method 3: Design Tools (Most Professional)

**Using Figma/Sketch:**
1. Create artboard: 1290 x 2796px
2. Import screenshot from simulator
3. Add text overlays
4. Add gradient backgrounds
5. Export as PNG

**Pros:** Professional look, easy to iterate
**Cons:** Takes more time

## üìä SCREENSHOT CHECKLIST

Before uploading to App Store Connect:

### Technical Requirements
- [ ] Correct dimensions (1290 x 2796 for iPhone 15 Pro Max)
- [ ] PNG or JPEG format
- [ ] Under 8 MB per file
- [ ] RGB color space (not CMYK)
- [ ] No transparency

### Content Requirements
- [ ] No placeholder or dummy data
- [ ] Realistic user names and content
- [ ] No personal information (phone numbers, emails visible)
- [ ] Professional looking data
- [ ] UI looks polished and clean

### Design Quality
- [ ] Text overlays readable
- [ ] Text doesn't cover important UI
- [ ] Consistent styling across all screenshots
- [ ] High contrast for readability
- [ ] Green brand color (#4CAF50) visible

### Legal Requirements
- [ ] No competitor logos (unless using their service)
- [ ] No copyrighted images (use own photos)
- [ ] Store logos used appropriately (Costco, Sam's Club)
- [ ] No misleading claims
