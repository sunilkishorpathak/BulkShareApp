**Move from Desktop to repo:**
```bash
mv ~/Desktop/*.png images/raw/
```

### 3. Add Text in Canva (60 min)

**Go to:** https://www.canva.com

**For each screenshot:**
1. Create design: 1290 x 2796 pixels
2. Upload screenshot from `images/raw/`
3. Add black rectangle (70% opacity) at top
4. Add white text (see text below)
5. Download PNG ‚Üí save to `images/final/`

### 4. Upload to App Store (10 min)

**Go to:** https://appstoreconnect.apple.com

1. My Apps ‚Üí BulkMates ‚Üí Version 1.0
2. Scroll to "App Store Screenshots"
3. Select "iPhone 6.7" Display"
4. Upload all 6 from `images/final/`
5. Save
